3.1:Features append:

Feature class append supports multiple to one addition of any data source from SHP, GDB, MDB, SDE to SHP, GDB, MDB, and SDE. It controls whether the source data and target data are allowed to have field structure differences or filter through test or no test parameters. Test mode is forced to limit, and the source data and target data format must be consistent, otherwise it cannot be run; no test mode will filter field mapping, and only the source data conforming to the mapping rules will be appended.
