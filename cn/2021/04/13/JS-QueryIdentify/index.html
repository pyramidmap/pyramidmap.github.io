PyramidMap Javascript API 地图查询

PyramidMap的地图查询功能，支持各种地图平台接口及其地图资源，例如ArcGIS及其online在线地图资源和本地托管的地图数据，GeoServer托管地图、Google Map、OSM、MapBox、天地图、高德/百度地图等在线资源。同时支持属性数据查询和空间地理位置查询，支持大数据量并发请求，无论是在线地图还是离线地图，都可以精准定位，并与用户自有的业务数据进行整合。可以根据用户服务器性能及网络负载情况，调整请求响应的返回值，确保地图展示的平滑稳定和数据的准确性。

坐标系兼容及准确的定位

PyramidMap内部集成了对不同坐标系的转换及位置校准，包括GCS空间坐标系统和PCS投影坐标系统，并针对高德/百度等特殊的坐标定义，也进行了兼容处理，实现精准定位。

属性数据查询

PyramidMap地图查询接口支持多数据源图层的属性查询，解析图层数据结构，支持多字段的联合查询，精准定位，并同步显示定位对象的详细信息。

空间数据查询

PyramidMap支持图层图形化空间数据查询。例如在城市管网地图上，可按照压力值及流向进行过滤查询，也可以按照城区或社区范围进行空间查询。
